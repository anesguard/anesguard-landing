# Anesguard Landing Page

AI 기반 마취 사전 평가 솔루션 Anesguard의 랜딩페이지.

## 개요

- **구조**: HTML + JS 싱글파일 (`index.html`)
- **폰트**: Pretendard (CDN)
- **파티클 효과**: 순수 Canvas API (라이브러리 의존성 없음)
- **반응형**: 모바일/데스크탑 대응

## 파일 구조

```
index.html          # 메인 랜딩페이지
assets/logo.svg     # 로고 SVG
docs/guide.md       # 카피 & 디자인 톤 기획
docs/particles.md   # 파티클 효과 기술 명세
poc-particles.html  # 파티클 효과 기술검증 페이지
```
